Agriculture On Smart Way

🚀 Live Demo 📦 Repository

📌 Project Overview

Agriculture On Smart Way is a modern web platform that connects farmers, suppliers, and consumers to streamline agricultural operations and improve decision-making.

  • Improve crop yields and resource efficiency
  • Real-time insights using IoT sensor data
  • Simplified supply-chain and marketplace operations

🛠 Tech Stack

  • Frontend: React, TypeScript, Vite, Tailwind CSS
  • Backend: Node.js, Express
  • Database: MongoDB
  • IoT: Arduino
  • DevOps: Docker, CI/CD

📂 Project Structure

  • client/ — React frontend
  • server/ — TypeScript API server
  • lib/ — Arduino / IoT code

⚙ Requirements

  • Node.js v18+
  • npm or yarn
  • MongoDB (local or cloud)

✨ Features

  • Real-time IoT telemetry
  • Analytics & crop recommendations
  • Role-based access (farmer, supplier, admin)
  • Dashboard for irrigation & monitoring
